# 🚀 Dynamic To-Do List (Modern UI)

## 📌 Project Overview

This is a modern and interactive To-Do List web application built using HTML, CSS, and JavaScript.
It allows users to efficiently manage daily tasks with a clean and responsive UI.

---

## ✨ Features

* ✅ Add new tasks dynamically
* ✏️ Edit existing tasks
* ❌ Delete tasks
* ☑️ Mark tasks as completed
* 📊 Task counter (Total & Completed)
* 🎨 Modern UI (Glassmorphism Design)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Modern UI + Animations)
* JavaScript (DOM Manipulation)
